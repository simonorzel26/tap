# bun

A TypeScript library scaffolded by init-npm-package.

## Installation

```bash
npm install bun
```

## Usage

```typescript
import { add, subtract, multiply, divide } from 'bun';

console.log(add(1, 2)); // 3
console.log(subtract(5, 2)); // 3
console.log(multiply(3, 4)); // 12
console.log(divide(10, 2)); // 5
```

## Development

```bash
npm install
npm run build
npm test
```
